    Odoo (boeking + betaling)

    Odoo Event Planner verwerkt boekingen én betalingen. Kies per sessie "Odoo" als betaalmethode.

    Mollie (workshop-betalingen)

    Workshop-betalingen verlopen nu volledig automatisch via Mollie — een klant die een sessie boekt, krijgt meteen een betaallink. Dit vereist geen instelling hier: de Mollie API-sleutel staat als omgevingsvariabele (MOLLIE_API_KEY) in Vercel, nooit in de website zelf.
