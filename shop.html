Verbinding met Odoo controleren…
⚙️ Odoo koppeling instellen

Bewerk odoo-config.js en vul je Odoo-instantie in:
— url: bijv. https://jouwbedrijf.odoo.com
— db: naam van je database
— apiKey: maak aan via Odoo → Instellingen → Technisch → API-sleutels

Zorg ook dat CORS ingeschakeld is op je Odoo-server (System Parameters → web.cors.origins = *).
Producten moeten de vlag "Can be Sold" hebben. Wanneer de verbinding werkt, worden producten automatisch geladen.
